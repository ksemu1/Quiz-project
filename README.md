# Интерактивная викторина (Quiz)

## Описание проекта
Интерактивная викторина создана с использованием HTML, CSS и JavaScript.

## Возможности
- 10 вопросов
- Подсчёт баллов
- Подсветка правильных и неправильных ответов
- Прогресс-бар
- Экран результатов
- Кнопка «Начать заново»

## Структура проекта
quiz-project/
- index.html
- css/style.css
- js/main.js
- README.md

## Запуск проекта
1. Открыть папку проекта
2. Запустить файл index.html в браузере
